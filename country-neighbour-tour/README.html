Tour Advice App

Tour Advice App(TAA) implements a Rest API to calculate the cost of going on a holiday to all surrounding countries given a budget. It makes use of geonames and the exchangeratesapi.

Usage

To use TAA you:

  • Go to http://localhost:8080/
  • Login to Googles OAuth Service
  • Go the following sample url: http://localhost:8080/tourAdvice?totalBudget=1200&budgetPerCountry=100&startingCountryCode=BG&currency=EUR
  • TotalBudget - the total budget for the trip
  • BudgetPerCountry - the budget for each country
  • StartingCountryCode - the countryCode for the country whose surrounding countries you want to visit.
